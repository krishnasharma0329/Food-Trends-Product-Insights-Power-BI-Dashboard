📊 Food Trends & Product Insights | Power BI Dashboard

An interactive Power BI dashboard that analyzes food trends and product performance to generate meaningful business insights. This project focuses on identifying customer preferences, top-performing products, and sales patterns to support data-driven decision-making.

🚀 Features

📈 Interactive and dynamic dashboards

🍔 Analysis of food trends and customer preferences

🛍️ Product performance insights

📊 Sales and revenue visualization

🔍 Filter and drill-down capabilities

📌 Easy-to-understand visual storytelling

🛠️ Tech Stack

Tool: Power BI

Data Processing: Power Query Editor

Data Analysis: DAX (Data Analysis Expressions)
